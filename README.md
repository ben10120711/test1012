# HR 简历-岗位匹配度分析工具

## 项目介绍

这是一个面向 HR 人员的 Web 工具，能够分析求职者简历与岗位描述的匹配程度，输出 0-100% 的匹配度分数，并给出相应的建议。

## 功能特点

- **双输入支持**：支持直接输入或上传岗位描述（JD），支持上传简历 PDF 文件
- **多维度分析**：基于硬技能匹配、经验匹配、教育背景、软技能与附加项四个维度进行分析
- **权重可调整**：HR 可以在界面上微调各维度的权重
- **详细的匹配结果**：显示总匹配度分数、各维度得分明细，以及相应的建议
- **直观的视觉展示**：使用环形进度条展示匹配度

## 技术栈

- **后端**：Python 3.10+
- **前端**：Streamlit
- **PDF 提取**：PyPDF2
- **文本分析**：jieba（中文分词）、scikit-learn（TF-IDF）

## 安装步骤

1. 确保已安装 Python 3.10 或更高版本
2. 克隆或下载本项目到本地
3. 进入项目目录
4. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用说明

1. 运行应用：
   ```bash
   streamlit run app.py
   ```
2. 在浏览器中打开生成的 URL
3. 按照界面提示：
   - 输入或上传岗位描述（JD）
   - 上传简历 PDF 文件
   - 调整各维度的权重（可选）
   - 点击「开始分析匹配度」按钮
4. 查看分析结果，包括总匹配度、各维度得分明细和相应的建议

## 注意事项

- 仅支持文字版 PDF 简历，扫描件图片无法解析
- 支持中英文混合文本，默认以中文为主
- 匹配度计算基于关键词匹配，可能存在一定的局限性
- 建议结合人工判断进行最终决策

## 匹配度计算逻辑

1. **文本提取**：从 PDF 简历和 JD 中提取纯文本
2. **关键词抽取**：对 JD 进行 TF-IDF 提取 top 20 个代表性关键词
3. **关键词分类**：将关键词分为硬技能词、经验词、教育词、软技能词
4. **各维度得分计算**：
   - 硬技能匹配：计算简历中出现的硬技能关键词数量 / JD 中硬技能关键词总数
   - 经验匹配：基于工作年限、职位名称相似度、项目描述相似度
   - 教育背景：基于学历层次、专业对口性
   - 软技能：计算简历中出现的软技能关键词数量 / JD 中软技能关键词总数
5. **总匹配度**：各维度得分乘以权重后求和

## 示例

1. 输入岗位描述，例如：
   ```
   职位：数据分析师
   职责：
   - 负责公司数据的收集、整理和分析
   - 制作数据报表，提供数据支持
   - 优化数据处理流程
   要求：
   - 本科及以上学历，计算机或相关专业
   - 2年以上数据分析经验
   - 熟悉 Python、SQL、Excel
   - 良好的沟通能力和团队合作精神
   ```

2. 上传简历 PDF 文件

3. 查看分析结果，例如：
   - 总匹配度：85%
   - 建议：✅ 建议邀约线下面试
   - 各维度得分：硬技能 90%，经验 80%，教育背景 100%，软技能 85%
